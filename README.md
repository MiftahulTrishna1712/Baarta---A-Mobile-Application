# Baarta---A-Mobile-Application
Baarta is an Android-based mobile application designed to provide real-time flood information, donation assistance, and emergency support for affected communities in Bangladesh.

 Features
🌊 Flood Information
   - Get real-time flood updates by district
   - Safety messages and alerts
   - Data fetched from Firebase
💰 Donation Calculator
   - Calculate cost of relief items (rice, water, etc.)
   - Dynamic pricing from database
📍 Donation Centers
  - View verified donation centers
  - Includes name and operating hours
🤖 Chatbot Assistant
  - Simple AI-based response system
  - Helps users navigate app features
🔐 Admin Panel
  - Secure login for administrators
  - Update flood data, prices, and centers
🧠 System Architecture
  - Frontend: Android (Java + XML)
  - Backend: Firebase Realtime Database
  - Authentication: Firebase Auth
  - Data Source: Firebase / External API (optional)
🔥 Technologies Used
  - Java
  - Android Studio
  - XML (Material UI)
  - Firebase Realtime Database
  - Firebase Authentication
